# <a name="declare-base-class"></a>Declarar una clase base
Puede declarar una clase de Windows PowerShell como un tipo base para otra clase de Windows PowerShell.

```PowerShell
class bar
{
   [int]foo() 
       {
           return 100500
       }
}

class baz : bar {}

[baz]::new().foo() # return 100500
```
